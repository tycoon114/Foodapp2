# 🍽️ FoodApp
FoodApp2는 사용자가 입력한 텍스트를 기반으로 음식 추천을 제공하는 안드로이드 애플리케이션입니다. 음식 데이터는 별도의 CSV(`food.csv`) 파일에서 불러오며, 앱 내 알고리즘은 텍스트 분석과 가중치 계산을 통해 최적의 음식을 추천합니다.

## 📌 프로젝트 개요
- **개발 환경**: Android Studio  
- **언어**: Java  
- **핵심 로직**: `Algorithm.java`  
- **데이터 관리**: 음식 데이터를 정적 구조로 저장 후 알고리즘을 통해 검색/출력  
- **참여 인원**: 2명 

---

## ⚙️ 주요 기능

- **CSV 데이터 로드**  
  - `food.csv` 파일을 읽어 음식 이름, 종류, 유형, 맛 정보를 배열에 저장  

- **텍스트 기반 분석**  
  - 사용자가 입력한 문자열에서 **인사, 음식 종류, 음식 유형, 맛 유형**을 추출  
  - 예: "매운 한식 먹고 싶어" → `매운맛`, `한식`  

- **가중치 기반 추천 알고리즘**  
  - 음식 종류 일치 시 +2점  
  - 음식 유형 일치 시 +1점  
  - 맛 유형 일치 시 +1점 (단, "안"이 붙은 경우 -1점)  
  - 최종적으로 가장 높은 점수를 받은 음식 후보 중 랜덤으로 하나 추천  

---

## 🧩 알고리즘 상세

### 1. 인사 처리
- 입력에 "안녕", "하이", "반가" 등이 포함되면 "안녕하세요"를 출력에 추가  

### 2. 음식 종류 매칭 (+2점)
- 한식, 중식, 일식, 분식, 치킨, 피자, 족발, 보쌈 등 키워드 매칭  

### 3. 음식 유형 매칭 (+1점)
- 면, 국, 튀김, 밥, 빵, 고기, 생선, 간식 등 키워드 매칭  

### 4. 맛 유형 매칭 (+1점 / -1점)
- 단맛, 짠맛, 신맛, 쓴맛, 감칠맛, 지방맛, 매운맛  
- "안단맛", "안매운맛"처럼 부정 표현이 있으면 -1점 처리  

### 5. 추천 결과
- 모든 음식에 대해 가중치를 계산  
- 최고 점수를 받은 음식 후보 중 랜덤으로 하나 선택  
- 인사 메시지와 함께 최종 추천 결과 출력
